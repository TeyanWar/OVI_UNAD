VPN

Permite crear una red privada que se utiliza para conectar una o más computadoras mediante internet. Así se prevé que un tercero robe información confidencial. Generalmente se usa en las empresas

Antivirus

Herramienta de seguridad que ayuda a proteger una computadora contra virus informáticos como: Troyanos, gusanos, etc

Firewall por hardware

Sistema que permite proteger una computadora contra ataques piratas, mediante un filtro de paquetes de datos que andan por la red
